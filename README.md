NothingGPT
---------------------------
NothingGPT is an AI chatbot for the Linux terminal. Here is how to install it:
To install NothingGPT, just follow these commands:

1| apt update – This updates the list of available packages.

2| apt upgrade – This upgrades all packages on your system.

3| apt install git – This installs Git, which you will need to copy the repository.

4| git clone https://github.com/TreeByte403/NothingGPT – This copies the NothingGPT repository to your computer.

5| cd NothingGPT – This changes the directory to the NothingGPT folder. All necessary files are here.

6| chmod +x setup.sh – This makes the setup script executable.

7| ./setup.sh – This runs the setup script. It will automatically install everything. After that, you can enjoy using NothingGPT in your terminal.

------------------------------
To exit press control + c or control + d.
